作为团队技术先驱与架构负责人，主导全栈技术前瞻规划与落地实践，推动团队研发能力实现本质飞跃，大幅提升系统稳定性、并发能力与迭代效率，形成可复用、可扩展的技术体系与工程规范。
一、架构与技术栈全面现代化升级
● 主导后端技术体系深度演进：完成从传统 Eclipse JSP 迁移至 IDEA Maven 工程化体系；推动架构从 Spring Boot 单体升级为微服务架构；完成技术栈从 Spring Web 向 Spring WebFlux + R2dbc 响应式异步架构转型，解决高并发、状态管理、异步任务、复杂交互场景的性能瓶颈。
● 推动前端架构现代化：从 Layui 传统框架升级为 Vue 技术体系，落地前后端分离；全面普及 Vue3 Composition API、<script setup> 规范，并完成 Uniapp 跨端框架的 Vue3 适配，统一团队技术标准。
● 构建从账套、租户到组织架构的多租户权限与资源隔离体系，实现业务与权限的精细化、规模化管理。
二、研发效能与工程化体系建设
● 代码管理：推动从 SVN 集中式管理，逐步迁移至Github、云效 Git、腾讯 Coding 等现代化协作平台，规范分支与发布流程。
● 制品与镜像管理：搭建 Maven/NPM 私有仓库，部署 Harbor 镜像仓库，形成统一制品管理体系。
● CI/CD 持续集成：推动流水线从 Jenkins 逐步升级至云效 CI、GitHub CI/CD，实现自动化构建、测试与部署，研发效率显著提升。
● 云原生化实践：完成容器化落地（Docker、Podman），并逐步向 K8s 部署与持续交付方向演进，为高可用、弹性扩缩容奠定基础。
● 开发环境升级：推动团队从 Eclipse 全面切换至 IDEA，并普及 IDEA + WSL Linux 环境，引入 Cursor 等 AI 辅助工具，提升本地开发体验与效率。
